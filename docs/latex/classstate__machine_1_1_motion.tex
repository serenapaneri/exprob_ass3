This class should execute the movement of the robot between the various rooms of the cluedo game. If there are no new complete and consistent hypotheses, then the robot should keep going searching new hints in other rooms, and the movement of the robot is implemented thanks to the move\+\_\+base package. If, instead, there is actually a new complete and consistent hypothesis the robot can go to the oracle room, trying its guessing. There can be also the possibility that the robot never find the correct solution and, when there are no more rooms to be visited the game finish. So, there are three outcomes\+:
\begin{DoxyItemize}
\item enter room, if the robot is still collecting hints
\item go\+\_\+oracle, if the hypothesis formulated is complete and consistent
\item game\+\_\+finished, if there are no more rooms to visit. 
\end{DoxyItemize}
