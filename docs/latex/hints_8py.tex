Description\+: ~\newline
 In this node is implemented the hints reciever of the cluedo game. In particular, when the state\+\_\+machine commands to start recieving hints the hints are first by selecting the not malformed ones and those are then uploaded in the ontology. The malformed ones are simply discarded. In the meanwhile the I\+Ds of that hints recieved are stored and sent to the state machine.
