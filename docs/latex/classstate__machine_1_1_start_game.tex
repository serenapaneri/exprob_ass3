This class should initialized all the things needed to start the investigation of the cluedo game. Here the robot assumes the default position, thanks to moveit, the ontology is loaded, all the individuals of the game are uploaded, then disjoint from each other and feed to the reasoner. Then the rooms of the game are randomized. Here there is only one outcome that is motion, in order to make the robot go to the various room, searching for hints.
